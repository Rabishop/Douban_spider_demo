## 使用到的库包括以下

| 库      | 功能       |
| ------- | ---------- |
| bs4     | 文本分析   |
| re      | 正则解析   |
| urllib  | 服务器访问 |
| xlwt    | Excel存储  |
| sqlite3 | 数据库存储 |

## 启动脚本

配置好环境后，直接运行应该就可以了

```
 python spider.py
```
